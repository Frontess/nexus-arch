# Nexus Arch — лендинг

## Обзор

**Nexus Arch** — это адаптивная лендинг-страница, демонстрирующая современные решения.

Проект предусматривает:

- Гибкую и адаптивную вёрстку с минимальным разрешением 320px.
- Шапку (header), которая фиксируется в верхней части экрана при скролле с эффектом затемнения фона.
- Подключение всех ресурсов (HTML, CSS, JavaScript, изображения) локально, без использования внешних библиотек и фреймворков.
- Гибкую структуру макета, позволяющую без проблем изменять количество текста, элементов списка или полей формы.
- Контактную форму, готовую к работе: каждое поле имеет корректный атрибут `name`.
- Динамические эффекты (анимации, hover-эффекты), улучшающие пользовательский опыт.

## Особенности

- **Адаптивная вёрстка:**  
  Страница корректно отображается на любых устройствах — от смартфонов с разрешением 320px до больших мониторов.

- **Локальные ресурсы:**  
  Все файлы (HTML, CSS, JS, изображения) находятся в проекте и подключены локально, что обеспечивает автономную работу сайта.

- **Гибкий макет:**  
  При изменении количества текстовых элементов, списков или полей формы вёрстка остаётся стабильной и не ломается.

- **Прозрачный контактный функционал:**  
  Контактная форма имеет правильно заданные поля (каждое поле – это input с атрибутом `name`), готовые для последующей интеграции с бэкендом.

- **Фиксированная шапка с динамикой:**  
  Шапка сайта остаётся на верхней части страницы при прокрутке, при этом применяется эффект затемнения фона для лучшей читаемости.

### Технологии:

HTML5 | CSS3 | JavaScript (ES6)

### Как запустить проект локально

Клонируйте репозиторий:

```bash
git clone https://github.com/Frontess/nexus-arch.git
```

Перейдите в директорию проекта:

```bash
cd nexus-arch
```

Запустите локальный сервер: Вы можете использовать простой сервер, например, http-server:

```bash
npm install -g http-server
http-server --cors
```

Откройте страницу в браузере: Перейдите по адресу http://localhost:8080 (или указанному порту).
